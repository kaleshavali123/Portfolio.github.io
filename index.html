Grocery App

Designed with a clean, responsive layout using HTML and CSS, the app ensures a smooth experience across all devices. JavaScript powers the interactive features, allowing users to add, delete, and mark grocery items as purchased ‚Äî all without needing to reload the page.

User Authentication System

This real-time project is a secure user authentication system built using the Java Full Stack.

The system ensures secure access to the application, protecting user data and enforcing access permissions. It serves as a foundational component for any application that requires user login functionality.

Core Features:

  • User Registration: A registration page allows users to create an account by entering their personal details. The submitted data is validated and securely stored in the database.
  • Login Functionality: Users can log in by entering their credentials, which are verified against the stored data. If the credentials are incorrect, an "Invalid credentials" message is displayed.
  • Password Reset: Users can reset their password by providing their registered email and correctly answering a security question. Upon successful verification, they are allowed to set a new password, with a confirmation message: "Password changed successfully." If the verification fails, an error message such as "Something went wrong!" is shown.
  • Responsive Design: Fully optimized for mobile, tablet, and desktop use.

Technologies Implemented:

  • HTML5: Structures the application using semantic elements.
  • CSS3: Enhances the visual presentation with sleek, adaptive styling.
  • Java: Implements the core backend logic for the user authentication system, including handling HTTP requests, processing login and registration data, validating user credentials, encrypting passwords, generating tokens or sessions, and communicating with the SQL database to manage user records. It ensures secure and efficient execution of authentication workflows.
  • SQL: SQL manages user data and credentials, enabling secure login, registration, and role-based access in the authentication system.
